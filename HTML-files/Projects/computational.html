Factor of Safety Analysis GUI for Aerospace Structures

This project developed a graphical user interface (GUI) to compute the Factor of Safety (FoS) for aerospace structures, such as spacecraft panels, under various loads. Built with Python (using Tkinter or PyQt), the GUI allows users to input material properties and load conditions, calculating FoS as FoS = ultimate stress / working stress per NASA standards (FoS ≈ 1.5–2.0). The result below shows the GUI displaying computed FoS, stress distributions, and material data. A collaborative effort by Aayusha Singh (GUI development, visualization) and Amrit Roy (algorithm design, validation).

Web Scraping for Aerospace Data Automation

This project automated the extraction of aerospace material properties and safety standards from NASA’s Technical Reports Server (NTRS) using Python (BeautifulSoup, Scrapy). Over 100 entries were scraped into CSV and JSON formats, adhering to ethical practices (e.g., robots.txt compliance). Results showcase structured data outputs. A collaborative effort by Aayusha Singh (scraper development, data processing) and Amrit Roy (data validation, automation scripting).
