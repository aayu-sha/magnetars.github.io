Factor of Safety Analysis GUI

This project developed a graphical user interface (GUI) to calculate the Factor of Safety (FoS) for structural components under various loads. Built with Python (Tkinter), the GUI allows users to input material properties and load conditions, computing FoS as FoS = ultimate stress / working stress. The result below shows the GUI displaying computed FoS, stress values, and input parameters. A collaborative effort by Aayusha Singh (GUI development, visualization) and Amrit Roy (algorithm design, validation).

Web Scraping for Data Automation

This project automated the extraction of structured data from public websites, such as product details or public records, using Python (BeautifulSoup, Scrapy). Over 100 entries were scraped into CSV and JSON formats, adhering to ethical practices (e.g., robots.txt compliance, rate limiting). Results showcase the scraped data in structured formats. A collaborative effort by Aayusha Singh (scraper development, data processing) and Amrit Roy (data validation, automation scripting).
